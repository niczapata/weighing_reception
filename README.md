## Module Name
Weighing Reception

## Description
Custom module for weighing reception management in Odoo 16. This module provides functionality to create and manage product receptions with integrated weighing capabilities, allowing users to efficiently track and record incoming products with their corresponding weights.

## Features
- Create and manage receptions
- Add products to receptions with weight tracking
- Record product weights during reception process
- Generate reports for received products
- Integration with Odoo 16 inventory system

## Installation
1. Copy the module to your Odoo addons directory
2. Restart your Odoo server
3. Update the apps list or reinstall the module
4. Install the module from the Apps menu

## Usage
1. Navigate to the Weighing Reception menu
2. Create a new reception
3. Add products to the reception
4. Enter the weight for each product
5. Confirm and save the reception

## Requirements
- Odoo 16.0
- Python 3.x
- Standard Odoo dependencies
